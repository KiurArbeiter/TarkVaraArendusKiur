LOWER CASE

Visual Studio IDE on loominguline stardiplatvorm, mida saate kasutada koodi redigeerimiseks, silumiseks ja koostamiseks ning seejärel rakenduse avaldamiseks.
Visual Studio sisaldab lisaks tavalisele redaktorile ja silurile, mida enamik IDEsid pakub, kompilaatorit,
koodikomplekteerimisvahendeid, graafilisi kujundajaid ja palju muid funktsioone, mis parandavad tarkvaraarenduse protsessi.


IntelliJ IDEA: See on veel üks populaarne integreeritud arenduskeskkond, mis pakub alumise taseme CASE-funktsioone.
See on disaninitud Java jaoks, aga töötab ka teiste keeltega. IntelliJ sisaldab lisandeid nagu:
Smart Code Assistance, Advanced Refactoring, Code Inspection, Version Control Integration, Database Tools jne


Git on tasuta ja avatud lähtekoodiga hajutatud versioonihaldussüsteem, mis on loodud selleks,
et kiirelt ja tõhusalt hallata kõike, alates väikestest kuni väga suurte projektideni.
See edestab selliseid SCM-vahendeid nagu Subversion, CVS, Perforce ja ClearCase selliste funktsioonidega nagu
odav kohalik hargnemine, mugavad staging-alad ja mitu töövoogu.
